# Security Policy

See the canonical version in the repository: [SECURITY.md](https://github.com/FasterApiWeb/laughing-chainsaw/blob/main/SECURITY.md).

## Quick summary

- Report vulnerabilities via [GitHub Security Advisories](https://github.com/FasterApiWeb/laughing-chainsaw/security/advisories/new) (private)
- Do **not** open public issues for exploitable bugs
- Never commit `key.hex`, `.env`, or health capture logs
